# StableWein
Package for evaluating perturbative unitarity (UNI) and bounded-from-below (BFB) conditions in a Z2 x Z2-symmetric three-Higgs-doublet model

- ["StableWein.nb"](https://github.com/jurciukonis/StableWein/blob/main/StableWein.nb): main package file.

- The binary files generated for compiled functions are provided in the ["Binaries"](https://github.com/jurciukonis/StableWein/blob/main/Binaries) directory.

- Documentation is available in the ["Documentation"](https://github.com/jurciukonis/StableWein/blob/main/Documentation) directory.

- Trained neural networks are stored in the ["Nets"](https://github.com/jurciukonis/StableWein/blob/main/Nets) directory.

- For further details and documentation, see Ref. [[1](https://arxiv.org/abs/2603.XXXXX)].
